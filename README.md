# ポモドーロ・タイマー (Web MVP)

- 25分作業 + 5分休憩の1サイクルを自動実行し、終了したら停止します。
- スタート／一時停止／リセットが可能です。
- 終了時にシンプルな電子音を鳴らします（WebAudio）。
- カウントダウン中に表示する画像／GIFをアップロードしてローカル保存できます（localStorage）。

## 使い方
1. `index.html` をブラウザで開きます（ローカルでOK）。
2. 「スタート」でカウント開始。作業 → 休憩の順に1回だけ自動で切り替わります。
3. 画像やGIFを選ぶとプレビューされ、次回以降も同じブラウザで保持されます。
4. 「画像をクリア」でローカル保存を削除できます。

## 今後の拡張方針

- 複数サイクル、ポモドーロの自動ループ

- iOS 版（SwiftUI）への移植。ローカル通知とLive Activity対応
